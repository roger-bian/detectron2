### Folder Structure
```
_data/
└── {dataset_name}/
    ├── labelme/    ## <---- manually annotated
    │   ├── train/
    │   └── val/
    └── coco/       ## <---- auto-generated by labelme2coco tool
        ├── train_{version}/
        │   ├── JPEGImages/
        │   ├── Visualization/
        │   └── annotations.json
        └── val_{version}/
            ├── JPEGImages/
            ├── Visualization/
            └── annotations.json
```

### Usage in training
```
python train.py \
    --dataset_name {dataset_name} \
    --version {version}
```
